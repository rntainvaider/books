ECMAScript (ES) — это стандартизированная спецификация языка программирования, на основе которой построен JavaScript. По сути, это набор правил, определяющих синтаксис, типы данных и другие основные элементы языка.

## Типы данных

- Примитивный тип данных (простой тип)
1. string
2. number
3. boolean
4. null - значение пусто или значение не известно.
5. undefined - значение не было присвоено.
6. symbol
7. bigint - большое число 2^53 - 1


- Непримитивный тип данных (сложный тип)
1. object

`typeof()` - функция определения типа данных.

JavaScript имеет динамическую типизацию, то есть если переменная имеет тип данных string, то можно поменять на тип данных number.

Бывает явное `Number('3')` и неявное `+'3'` преобразование типов. Лучше всего использовать явное.

# PHP

PHP - это слаботипизированный язык.

Слабая типизация языка означает, что строку и число можно сложить и получить число.

Объявление переменной начинается с `$hello`. Пример использования `print_r($hello)`.

## Именование переменных

- kebab-case - составные части переменной разделяются дефисом (my-super-var).
- snake_case - для разделения используется подчеркивание (my_super_var).
- CamelCase - каждое слово в переменной пишется с заглавной буквы (MySuperVar).
- loserCamelCase - каждое слово в переменной пишется с заглавной буквы, кроме первого (mySuperVar).

## Типы данных

1. string - строки.
2. int - целые числа.

## Магические константы

- __LINE__ — содержит текущую строку файла, в котором она используется.
- __FILE__ — путь до текущего файла.
- __DIR__ — путь до директории, в которой находится текущий файл.

## Интерполяция

```
<?php

$firstName = 'Joffrey';
$greeting = 'Hello';

// Обратите внимание на ограничители строки — это двойные кавычки
// Интерполяция не работает с одинарными кавычками
print_r("{$greeting}, {$firstName}!"); // => Hello, Joffrey!
```

## Heredoc

```
<?php

$str = <<<EOT
Пример строки,
охватывающей несколько строк,
с использованием heredoc-синтаксиса.
Здесь не нужно экранировать ни одинарные ', ни двойные " кавычки.
EOT;
print_r($str);
// => Пример строки,
// => охватывающей несколько строк,
// => с использованием heredoc-синтаксиса.
// => Здесь не нужно экранировать ни одинарные ', ни двойные " кавычки.
```

# REACT

## Настройка styles.module.css React + TypeScript

- Нужно создать файл `src/global.d.ts` и прописать в нем:

```
declare module '*.module.css' {
  const classes: { [key: string]: string };
  export default classes;
}
```

## Настройка svg формата React + TypeScript

- Нужно создать файл `src/global.d.ts` и прописать в нем:

```
declare module "*.svg" {
  import React from "react";
  export const ReactComponent: React.FC<React.SVGProps<SVGSVGElement>>;
  const src: string;
  export default src;
}
```

#  Список проектов python для обучения

Уровень 1: Начинающий (Основы Python, CLI)
1. Калькулятор

Описание: Консольное приложение, которое принимает два числа и оператор (+, -, *, /) как аргументы командной строки или через ввод пользователя и выводит результат.

Что изучите: Базовый синтаксис, арифметические операции, условные операторы (if/elif/else), работу с аргументами командной строки (sys.argv) или функцией input().

2. Генератор случайных чисел ("Угадай число")

Описание: Программа загадывает случайное число в заданном диапазоне (например, от 1 до 100). Пользователь вводит свои догадки, а программа подсказывает "больше" или "меньше", пока число не будет угадано.

Что изучите: Модуль random (randint), циклы (while), сравнение чисел.

3. Простой шифратор/дешифратор текста (Шифр Цезаря)

Описание: Программа, которая сдвигает каждую букву в тексте на фиксированное количество позиций в алфавите. Например, сдвиг на 3: 'A' -> 'D', 'Z' -> 'C'. Должны быть режимы как шифрования, так и дешифрования.

Что изучите: Работу со строками, циклы, методы ord() и chr() для конвертации символов в коды и обратно.

4. Список дел (To-Do List) для командной строки

Описание: Приложение позволяет добавлять, просматривать, удалять и отмечать задачи как выполненные. Список задач должен сохраняться между запусками программы в файл (JSON или CSV).

Что изучите: Работу со списками и словарями, чтение/запись файлов (json.load()/json.dump()), базовое управление состоянием приложения.

5. Конвертер валют с фиксированными курсами

Описание: Пользователь вводит сумму, исходную валюту и целевую валюту. Программа использует заранее заданный словарь с курсами (например, {'USD': 1.0, 'EUR': 0.85, 'RUB': 90.0}) и возвращает конвертированную сумму.

Что изучите: Словари, функции для инкапсуляции логики конвертации, форматирование вывода.

6. Генератор паролей

Описание: Программа создает случайные пароли заданной пользователем длины. Можно добавить опции: использовать ли цифры, специальные символы, прописные и строчные буквы.

Что изучите: Модуль random (функция choice или choices), модуль string для готовых наборов символов (string.ascii_letters, string.digits).

7. Игра "Камень, Ножницы, Бумага"

Описание: Пользователь делает свой ход (вводит 'к', 'н' или 'б'), программа генерирует случайный ход. Определяется победитель по правилам: камень побеждает ножницы, ножницы побеждают бумагу, бумага побеждает камень.

Что изучите: Условные операторы, случайный выбор, базовую логику игры.

8. Парсинг простого текстового файла

Описание: Программа принимает путь к текстовому файлу (.txt) и подсчитывает базовую статистику: количество слов, строк, символов (с пробелами и без), самое частое слово.

Что изучите: Чтение файлов (open(), read()), строковые методы (split(), strip()), работу с коллекциями (collections.Counter для подсчета слов).

9. Простой чат-бот для командной строки

Описание: Программа, которая ведет диалог с пользователем, отвечая на заранее заготовленные фразы по шаблону (например, на "Привет" -> "Приветствую!", на "Как дела?" -> "Отлично, а у тебя?"). Может заканчивать диалог по команде "Пока".

Что изучите: Циклы, условные операторы, базовые строковые методы.

10. Бинарный поиск по отсортированному списку

Описание: Реализация алгоритма бинарного поиска. Программа принимает отсортированный список и целевой элемент. Алгоритм repeatedly делит список пополам, пока не найдет элемент или не докажет его отсутствие.

Что изучите: Фундаментальный алгоритм, рекурсию или итерацию, работу с индексами списка.

11. Таймер обратного отсчета

Описание: Программа, которая принимает количество секунд и начинает обратный отсчет в реальном времени, выводя оставшееся время в консоль каждую секунду. По окончании выводит сигнал (например, звуковой).

Что изучите: Модуль time (sleep), форматирование вывода.

12. Генератор мемов

Описание: Приложение, которое берет базовое изображение (шаблон мема) и накладывает на него верхний и нижний текст, имитируя популярные мемы. Результат сохраняется как новое изображение.

Что изучите: Библиотеку Pillow (PIL) для работы с изображениями: открытие, создание объекта для рисования, настройка шрифта, рисование текста.

13. Валидатор email-адресов

Описание: Программа, которая проверяет строку на соответствие формату email-адреса с помощью регулярного выражения. Должна проверять наличие '@', домена и т.д.

Что изучите: Модуль re для работы с регулярными выражениями, составление паттернов.

14. Калькулятор расходов

Описание: Консольное приложение для учета личных финансов. Позволяет вводить расходы по категориям (еда, транспорт), сохранять их в файл и выводить общую сумму трат или по категориям.

Что изучите: Структуры данных для хранения записей, работу с файлами (json для сложных структур), агрегацию данных.

15. Поиск анаграмм в списке слов

Описание: Программа принимает список слов и находит среди них анаграммы (слова, составленные из одних и тех же букв в другом порядке, например, 'кот' и 'ток'). Группирует и выводит их.

Что изучите: Работу со строками (сортировка букв в слове), использование словарей для группировки.

16. Текстовый квест

Описание: Интерактивная история, где пользователь выбирает варианты действий, что влияет на развитие сюжета. Каждая "локация" — это текст и словарь с возможными действиями и следующими локациями.

Что изучите: Функции для описания локаций, циклы, словари для связывания состояний игры.

17. Простой веб-скрапер заголовка страницы

Описание: Программа, которая принимает URL, загружает HTML-код страницы и извлекает из него текст внутри тега ```<title>```.

Что изучите: Библиотеку requests для HTTP-запросов, BeautifulSoup4 для парсинга HTML.

18. Консольные "Крестики-Нолики"

Описание: Реализация игры для двух игроков на поле 3x3. Игроки по очереди вводят координаты (строка, столбец). Программа отрисовывает поле в консоли после каждого хода и проверяет условия победы.

Что изучите: Вложенные списки для представления игрового поля, сложную логику проверки победных комбинаций.

19. Генератор историй (Mad Libs)

Описание: Программа с заранее заготовленным шаблоном текста с пропусками. Она запрашивает у пользователя слова (существительные, прилагательные, глаголы), подставляет их в шаблон и выводит забавную получившуюся историю.

Что изучите: Строковые методы, форматирование строк (f-strings, .format()).

20. Утилита для резервного копирования папки

Описание: Скрипт, который принимает путь к папке и создает ее zip-архив с timestamp в имени файла (например, backup_20231027_120000.zip).

Что изучите: Модули os для работы с путями, shutil для создания архивов, datetime для формирования имени.

30. Конвертер изображений

Описание: Утилита командной строки, которая принимает путь к файлу, целевой формат и опционально параметры (качество, размер) и конвертирует изображение (например, все PNG в папке в JPG).

31. Уведомитель о курсе криптовалют

Что изучите: Планировщики задач (schedule), работу с реальными финансовыми API, основы построения мониторинговых систем.

32. Простейший HTTP-сервер

Описание: Собственный веб-сервер, способный обрабатывать HTTP-запросы и возвращать статические файлы (HTML, CSS, изображения) или простые текстовые ответы.

Что изучите: Модули http.server и socketserver, базовое понимание протокола HTTP (запросы, ответы, заголовки).

33. Утилита для пакетного переименования файлов

Описание: CLI-утилита, которая рекурсивно обходит папку и переименовывает файлы по заданному шаблону (например, добавляет префикс, меняет расширение, приводят к lower case).

Что изучите: Модуль os и os.path для работы с файловой системой, argparse для гибких настроек.

34. Генератор QR-кодов

Описание: Приложение, которое принимает строку (ссылку, текст) и генерирует изображение QR-кода. Можно сделать как консольную утилиту, так и с графическим интерфейсом.

Что изучите: Kubernetes — ведущую систему оркестрации контейнеров, основы DevOps.

Описание: Написание с нуля веб-сервера, способного обрабатывать HTTP/1.1 запросы (GET, POST), обслуживать статические файлы и запускать простые WSGI-приложения. Реализация на низком уровне с использованием модуля socket.

Что изучите: Глубокое понимание протокола HTTP, сокеты, модель WSGI.

83. Высокопроизводительный кэширующий HTTP-прокси

Описание: Создание прокси-сервера, который принимает запросы от клиентов, перенаправляет их на целевые сервера и кэширует ответы. Кэширование должно быть асинхронным и эффективным. Можно добавать поддержку HTTPS.

Что изучите: Асинхронное программирование (asyncio, aiohttp), кэширование, сетевые технологии.

84. Система для проведения A/B тестов

Описание: Платформа, позволяющая продукт-менеджерам создавать A/B тесты (разные версии заголовка сайта, цвета кнопки). Система случайно распределяет пользователей по группам (A/B), собирает статистику и показывает, есть ли статистически значимая разница в их поведении.

Что изучите: Django/Flask, проектирование схемы БД для экспериментов, статистику (p-value, t-test) на scipy.

85. Собственный DSL (Domain Specific Language)

Описание: Проектирование и реализация простого языка для конкретной предметной области. Например, язык для описания бизнес-правил ("IF user.age > 18 AND order.sum > 1000 THEN APPLY_DISCOUNT(10)"). Включает парсер, лексер и интерпретатор.

Что изучите: Теорию компиляторов, парсер-комбинаторы или инструменты (PLY, Lark).

86. Модель для генерации текста (char-level RNN)

Описание: Обучение рекуррентной нейронной сети (RNN или LSTM) генерировать текст по символьно, имитируя стиль тренировочных данных (например, тексты Шекспира). На вход получает один символ и предсказывает следующий.

Что изучите: TensorFlow/PyTorch, рекуррентные нейронные сети, работу с последовательностями.

87. Транскриптор аудио в текст

Описание: Создание сервиса, который принимает аудиофайл (например, запись голоса) и возвращает его текстовую расшифровку. Можно использовать готовые модели (Whisper от OpenAI) или облачные API (Google Speech-to-Text) на начальном этапе.

Что изучите: Обработку аудиосигналов, библиотеки (SpeechRecognition), работу с глубокими моделями для ASR (Automatic Speech Recognition).

88. Система компьютерного зрения для отслеживания объектов

Описание: Приложение, которое получает видеопоток (с веб-камеры или файла) и в реальном времени выделяет и отслеживает движущиеся объекты (люди, машины), рисуя bounding boxes вокруг них. Можно использовать предобученные модели (YOLO, OpenCV tracker).

Что изучите: OpenCV для работы с видео, глубокое обучение для компьютерного зрения, модели детекции объектов.

89. Фреймворк для автоматического трейдинга с ML

Что изучите: ccxt, фреймворки для бэктестинга (backtrader, zipline), feature engineering для финансовых данных, интеграцию ML-моделей (scikit-learn, xgboost) в торговые стратегии.

Что изучите: Распределенные системы, сокеты для межпроцессного взаимодействия, хеширование, устойчивость к сбоям.

91. Реализация собственной простой реляционной СУБД

Описание: Создание с нуля упрощенной базы данных, которая понимает подмножество SQL (например, CREATE TABLE, INSERT, SELECT с WHERE). Включает парсер SQL, механизм хранения данных на диске (B-деревья или просто файлы) и исполнитель запросов.

Что изучите: Внутреннее устройство СУБД, структуры данных (B-деревья), парсинг.

92. Система для обработки потоковых данных (data pipeline)

Описание: Построение пайплайна, который потребляет поток событий (логи, клики) из Kafka, обрабатывает их (агрегирует, фильтрует) с помощью потокового процессора (например, Faust) и сохраняет результат в Elasticsearch для быстрого поиска или в другую БД.

Описание: Создание API на GraphQL поверх вашего Django/Flask приложения. GraphQL позволяет клиенту запрашивать только нужные ему поля и связанные данные в одном запросе, решая проблемы over-fetching и under-fetching REST API.

Что изучите: GraphQL, сравнение с REST, библиотеки (Graphene для Python).

94. ML-модель для предсказания временных рядов

Описание: Построение модели для прогнозирования значений на будущее на основе исторических данных (продажи, курс акций, температура). Использование классических моделей (ARIMA из statsmodels) или более современных (Facebook Prophet).

Что изучите: MLOps-практики, MLflow, DVC, FastAPI для создания эндпоинтов, мониторинг моделей в продакшене.

Описание: Интеграция Elasticsearch в ваше приложение (например, в блог). Настройка индексации новых статей при их сохранении и реализация быстрого и релевантного поиска по ним с учетом морфологии, синонимов.

Что изучите: Elasticsearch и его Python-клиент, индексацию данных, написание поисковых запросов.

Описание: Реализация ключевых компонентов алгоритма Raft (выборы лидера, репликация лога) для набора серверов. Это основа для построения отказоустойчивых распределенных систем (как у Consul, etcd).

Что изучите: Распределенные алгоритмы, консенсус, сокеты, многопоточность/асинхронность.

Что изучите: Работу в большой кодовой базе, командную разработку, Git (fork, pull request), коммуникацию с сообществом.

Балансировщик нагрузки (round-robin): Распределение HTTP-запросов между несколькими серверами. Технологии: проксирование TCP-соединений, пул соединений.

Система мониторинга (агент): Сбор метрик (CPU, RAM) и отправка на сервер. Технологии: кросс-платформенный код, сокеты, JSON.

Peer-to-peer файлообмен (децентрализованный): По типу Gnutella (очень упрощенно). Технологии: поддержание списка соседей, flooding-запросы.

Категория D: Графика, игры и симуляции
Визуализатор Мандельброта/фракталов: С увеличением и палитрой. Технологии: SFML/OpenGL, многопоточный расчет, комплексные числа.

Физический движок (2D): Столкновение кругов и прямоугольников с импульсами. Технологии: математика (векторы), закон сохранения импульса.

3D-визуализатор вокселей (типа Minecraft): Чанки, отсечение невидимых граней. Технологии: OpenGL, шейдеры (GLSL).

Графический редактор для пиксель-арта: Палитра, слои, инструменты. Технологии: Qt, работа с текстурами.

Категория E: Обработка данных и ML

Рекомендательная система (collaborative filtering): Похожие пользователи на основе оценок. Технологии: SVD (сингулярное разложение), библиотека Armadillo.

Генератор текстов (Markov chain): На основе частотности пар/троек слов. Технологии: std::unordered_map с кортежами в качестве ключей.
